# GodCat

> Make cats in Minecraft invincible

## Dependencies

- openjdk@21
- Forge 1.20.1

## Building

Then build the plugin:

```
$ ./gradlew build
```

## Installing

Copy the built GodCat jar file inside `build/libs/godcat-1.0.0.jar` to your `mods/` directory in your minecraft server's data folder.
