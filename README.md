# GodCat

> Make cats in Minecraft invincible

## Dependencies

- openjdk@21
- Paper/Spigot 1.21.3

## Building

First build spigot (takes a while):

```
$ make deps
```

Then build the plugin:

```
$ make build
```

## Installing

Copy the built GodCat jar file inside `target/` to your `plugins/` directory in your minecraft server's data folder. Then start the server and watch your cat live it's best, godmode-enabled life!
